# MSAT

AClinically Aligned Heterogeneous Graph Learning Framework for Predicting
 Adverse Drug Reactions Induced by Chinese Materia Medica

---

## Requirements

```bash
pip install -r requirements.txt
```

Required packages:
- PyTorch >= 1.10
- PyTorch Geometric >= 2.0
- NumPy
- Scikit-learn
- SciPy
